HTML

HyperText Markup Language — стандартный язык разметки для создания веб-страниц.

CSS

Cascading Style Sheets — язык стилей, используемый для описания внешнего вида документа, написанного на HTML.

JavaScript

Язык программирования, который позволяет создавать динамические и интерактивные элементы на веб-страницах.
